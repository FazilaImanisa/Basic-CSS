Materi CSS

CSS adalah kode untuk membuat styling untuk halaman web. CSS terdiri dari selector, property, dan value.

Peletakkan CSS ada 3 macam, yaitu Inline Style, Internal Style Sheet, External Style Sheet

Belajar membuat Web itu membuat hati menjadi senang

Selector ada 3 jenis yaitu nama tag HTML, id tag HTML, dan class tag HTML.
